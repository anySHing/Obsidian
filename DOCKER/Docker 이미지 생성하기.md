# Bash 익히기

Docker가 리눅스 기반이기 때문에 이미지를 생성할 때 Bash(Bourne-again shell)를 주로 사용한다. 그래서 이미지를 생성하기 전에 자주 사용하는 Bash 문법을 간단히 알아본다.

| 문법 | 설명 |
|:----:|:----:|
|  >   |      |
|  <   |      |
|  >>  |      |
|  2>  |      |
| 2>>  |      |
|  &>  |      |
| 1>&2 |      |
| 2>&1 |      |
|  \|  |      |
|      |      |
|      |      |
|      |      |
|      |      |
